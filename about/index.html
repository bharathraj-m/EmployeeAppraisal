CREDIT BASED PERFORMANCE APPRAISAL SYSTEM

This project aims at creating a healthy Superior-Subordinate Relationship. Each role in the project is provided with credentials for logging into the system portal. Based on the login credentials, every role is identified. Each role is provided with specific set of privileges. Administrator manages the overall system by adding or removing employees and assigns new projects, project manager and project priority. Project Manager is responsible for assigning subtasks with priorities and employees to that particular subtask. He monitors the progress of the project and approves or rejects the submissions made by the employees. Employees work on the allotted subtasks and report to the Manager via notifications. At the end of each subtask and project, credits are automatically calculated and accredited by the system based on the employeeâ€™s performance. This in turn creates a fair and transparent working environment in the organization thereby motivating every employee to achieve more on their part.
